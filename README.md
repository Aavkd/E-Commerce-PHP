# SaaS Agency Website

## Project Overview
A high-performance SaaS Agency website built with pure PHP and MySQL.

## Structure
- `api/`: JSON Endpoints
- `src/`: Core Logic (Models, Utils)
- `config/`: Database configuration
- `public/`: Frontend assets and pages

## Setup
1. Import `schema.sql` into your MySQL database `agency_db`.
2. Configure `config/database.php` (or `.env` if implemented) with your DB credentials.
3. Serve the `public/` directory.
